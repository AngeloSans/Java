## Design Patterns

This repository includes implementations of various design patterns in Java. Each pattern is demonstrated with clear examples and explanations to help you understand how and when to use them.

- Creational Patterns
  - Singleton
  - Factory Method
  - Abstract Factory
- Structural Patterns
  - Adapter
  - Composite
  - Decorator
- Behavioral Patterns
  - Observer
  - Strategy
  - Command
